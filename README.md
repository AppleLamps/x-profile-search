## ‚ö° Getting Started

### Prerequisites

- Node.js 18+
- xAI API Key

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/x-profile-search.git
   cd x-profile-search
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment**
   Create a `.env.local` file based on the example:
   ```bash
   cp .env.local.example .env.local
   ```
   Add your API key:
   ```env
   XAI_API_KEY=your_xai_api_key_here
   ```

4. **Run the application**
   ```bash
   npm run dev
   ```
